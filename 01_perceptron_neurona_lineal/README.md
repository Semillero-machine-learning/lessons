# Perceptrón / Neurona lineal (Fahrenheit ↔ Celsius)

Implementación de una neurona lineal simple para predecir temperatura en Celsius a partir de Fahrenheit, incluyendo normalización de datos, entrenamiento y visualización de resultados.

## Contenido
- `neurona_lineal_fahrenheit_celsius.ipynb` — notebook principal con la implementación, normalización y gráficas.
- `Perceptron.pdf` — material teórico de apoyo.
