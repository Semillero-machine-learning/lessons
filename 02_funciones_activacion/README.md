# Funciones de activación (ReLU, Leaky ReLU, Identidad, Tanh)

Exploración práctica de distintas funciones de activación: implementación, visualización y comparación de su comportamiento.

## Contenido
- `relu_leakyrelu.ipynb` — implementación y análisis de ReLU y Leaky ReLU (datasets, visualización, gradient check, conclusiones).
- `identidad_tanh.ipynb` — implementación y comparación de las funciones Identidad y Tanh, con widgets interactivos.
- `SemAct2.pdf` — material teórico de apoyo.
